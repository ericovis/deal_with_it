Deal With It!

A GIF generator service written in Python

Demo

Either insert and valid image URL or click on "Choose File" to load a picture from your computer then click on the "Deal with it!" button to see the API in action.

Api Docs

There is a simple REST api available at /api that receives a POST request and expects either an image encoded in a Base64 string or an url

Example payloads


{
    "url": "https://emagalha.es/images/me.jpg"  
}
                

{
    "image": "base64 string here"  
}
